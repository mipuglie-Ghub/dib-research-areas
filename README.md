# Research Areas mini-site (real content)

This mini-site is generated from your `Research_Areas_Revised_GCM2.docx` and is meant to be embedded in Google Sites via an iframe.

## Files
- `index.html` : overview with cards
- `*.html`     : one page per research area
- `style.css`  : styling (edit `--accent` to match dibernardogroup)
- `assets/`    : placeholder images (replace with your artwork)

## Deploy with GitHub Pages (recommended)
1. Create a new GitHub repo (e.g. `research-areas`).
2. Upload all files from this folder.
3. Enable Pages: Settings → Pages → Deploy from branch → `main` / root.
4. You get `https://<user>.github.io/<repo>/`

## Embed in Google Sites
- Insert → Embed → URL → paste `https://.../index.html`
- Resize frame height (start ~1400px and adjust).
